# Priya Investor Dialogue Script

Use this on the live meeting page between Maya Chen (founder, Meshwave) and Priya Iyer (Principal, Lattice Ventures).

The goal is to clearly trigger six capabilities — now with Exa live research fused in:

1. external-signal recall (Exa: what changed at Lattice since last time)
2. internal milestone recall
3. unresolved concern detection
4. suggested founder question
5. best partner recommendation (warm intro)
6. promised follow-up and memory capture
